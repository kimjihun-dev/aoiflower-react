# 아오이플라워 (AoiFlower)

플로리스트 아오이플라워의 공식 홈페이지입니다.

## 프로젝트 소개

- 프로젝트명: 플로리스트 아오이플라워 홈페이지 제작
- 개발 기간: 2024.01 ~
- 개발자: kimjihun

## 기술 스택

### Frontend
- React
- CSS
- Swiper

### 개발 환경
- Visual Studio Code
- Git

## 주요 기능

- 반응형 웹 디자인
- 이미지 슬라이더
- 모바일 최적화

## 실행 방법

```bash
npm install
npm run dev
```
